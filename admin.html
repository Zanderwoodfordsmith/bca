Import Coaches from CSV

CSV Format Requirements:

Your CSV file should have the following columns (in any order):

  • first_name - Coach's first name (required)
  • last_name - Coach's last name (required)
  • email - Email address (optional)
  • bio - Bio/description (optional)
  • profile_image - Profile image URL (optional)
  • phone - Phone number (optional)
  • website - Website URL (optional)
  • linkedin_url - LinkedIn profile URL (optional)
  • book_call_url - Book a call URL (optional)
  • gender - Gender (optional)
  • join_date - Join date in YYYY-MM-DD format (optional)
