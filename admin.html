Import Coaches from CSV

CSV Format Requirements:

Your CSV file should have the following columns (in any order):

  • first_name - Coach's first name (required)
  • last_name - Coach's last name (required)
  • email - Email address (optional)
  • bio - Bio/description (optional)
  • profile_image - Profile image URL (optional)
  • phone - Phone number (optional)
  • website - Website URL (optional)
  • linkedin_url - LinkedIn profile URL (optional)
  • book_call_url - Book a call URL (optional)
  • gender - Gender (optional)
  • join_date - Join date in YYYY-MM-DD format (optional)

Import Wins from CSV

CSV Format Requirements:

Your CSV file should have the following columns (in any order):

  • win_title - Title of the win (required)
  • coach_name - Coach's first and last name (optional - can be blank)
  • win_description - Description of the win (optional)
  • win_category - Category of the win (optional)
  • win_date - Date in YYYY-MM-DD format (optional)
  • show_on_wall - true/false to show on proof wall (optional, defaults to true)
  • priority - Priority level: high, medium, or low (optional, can be blank)
  • win_status - Status of the win: verified, pending, or rejected (optional, defaults to pending)

Note: Only win_title is required. Coach names will be looked up automatically if provided. If no coach is specified or coach is not found, the win will be imported without a coach assignment.
